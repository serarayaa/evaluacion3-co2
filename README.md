# Evaluación Parcial 3 — Análisis interactivo de emisiones de CO₂

Proyecto grupal para la asignatura **Programación para la Ciencia de Datos**.

## Objetivo
Analizar emisiones de CO₂ usando Python, Pandas, Plotly y un mini dashboard en notebook.

## Archivos Incluídos
- `Evaluacion_3_CO2_Google_Colab.ipynb`: notebook principal del análisis.
- `owid-co2-data.csv`: dataset utilizado.
- `Presentacion_Evaluacion_3.pptx`: presentación del trabajo.

## Visualizaciones incluidas
1. Gráfico de líneas: evolución mundial de emisiones de CO₂.
2. Gráfico de barras: top 10 países por emisiones en 2024.
3. Gráfico de dispersión: relación entre población y CO₂ total.
4. Visualización complementaria: Chile y comparación regional.
